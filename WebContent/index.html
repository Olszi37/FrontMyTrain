MyTrain - mała apka do wyszukiwania połączeń kolejowych


Aplikacja na wzór strony PKP Intercity, stworzona głównie na potrzeby nauki Spring, Hibernate etc.

Całość podzielona jest na dwie aplikacje: back-endową i front-endową.

Aplikacja wciąż w budowie.

Zastosowane technologie:

  • Spring
  • Maven
  • Hibernate
  • Apache POI
  • HTML
  • jQuery
  • CSS
